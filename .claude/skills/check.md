# Branch Check

Quick check of your current Git branch status.

## Usage

```bash
bash .claude/scripts/branch-check.sh
```

## What it shows

- Current branch name
- Uncommitted changes (if any)
- Commits ahead/behind remote
- Whether you need to push or pull

## When to use

- Before switching branches
- Quick sanity check before committing
- After finishing work to ensure everything is pushed
