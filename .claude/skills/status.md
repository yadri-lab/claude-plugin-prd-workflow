# PRD Status Check

Quick command to see all active PRDs and their current status.

## Usage

```bash
bash .claude/scripts/prd-status.sh
```

## What it shows

- Count of PRDs by status (In Progress, In Review, Completed)
- List of active PRDs with their branches
- Quick overview without opening files

## When to use

- Before starting new work to see what's active
- Quick status check during standup
- Planning which PRD to work on next
