# Thoughts Directory

This directory stores persistent memory artifacts from the PRD workflow:

- **research/**: Auto-generated research documents (from /code-prd Phase 1)
- **plans/**: Auto-generated implementation plans (from /code-prd Phase 2)
- **validation/**: Auto-generated validation reports (from /code-prd Phase 4)
- **explorations/**: Feature explorations (from /explore-prd)
- **debugging/**: Debug session logs (from /debugging)

These files survive context clearing and provide team knowledge base.
